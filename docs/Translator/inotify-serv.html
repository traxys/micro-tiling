Translator Service¶

inotify_serv.encode(string)[source]¶

Encode the string in the most frequent kanji encoding

inotify_serv.encrypt(string, gpg)[source]¶

Encrypt the string in pgp encoded using the context gpg

inotify_serv.listen(watch_dir)[source]¶

Listen for new files in watch_dir and reads them, translates the segments and forwards them

inotify_serv.send(host, segments, job_id, gpg)[source]¶

Sends the encoded segments to the next service by smtp throught host encrypted using the gpg context

inotify_serv.translation(segments, job_id, gpg)[source]¶

Creates replicas of the segments in the eight directions
